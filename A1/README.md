# Метод Монте-Карло

## Файлы

- `wide.txt` — для широкой области генерации точек
- `narrow.txt` — для узкой области генерации точек
- `graph(1-4).png` — графики сравнения ошибок для широкой и узкой областей

### Стабильность

- **Узкая область** обеспечивает гораздо более стабильные значения ошибки, даже при малом числе точек.  
- **Широкая область** демонстрирует значительные колебания ошибки при небольшом N.  

### Величина ошибки

- Для малых N (100–5000):  
  - Ошибка широкой области может достигать 25–30%  
  - Ошибка узкой области не превышает ~2%  
- Для N ≈ 100 000:  
  - Ошибка широкой области снижается до 1–2%  
  - Ошибка узкой области снижается до ~0.1%  

## Выводы

- Выбор области генерации точек критически влияет на точность метода Монте-Карло.  
- Использование **узкой области**, ограничивающей пересечение кругов, позволяет снизить дисперсию и достичь высокой точности быстрее.  
- **Широкая область** приводит к сильным колебаниям и медленной сходимости при малом N.  
- Метод Монте-Карло эффективен для приближённого вычисления сложных фигур, но важно корректно выбирать область генерации случайных точек для ускорения сходимости.
